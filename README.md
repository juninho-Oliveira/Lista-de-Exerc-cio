# Lista-de-Exerc-cio# Exercícios em Linguagem C

Este repositório contém a resolução de uma lista de exercícios de programação desenvolvidos na linguagem C. Os exercícios têm como objetivo praticar conceitos básicos de lógica de programação, entrada e saída de dados, estruturas condicionais, estruturas de repetição e manipulação de vetores.

## 📚 Sobre a atividade

A atividade consiste na implementação de diversos problemas utilizando a linguagem C. Cada exercício foi desenvolvido com foco na prática dos fundamentos da programação e na adaptação da lógica de programação para a sintaxe da linguagem.

## 🧠 Conteúdos praticados

* Entrada e saída de dados (`printf` e `scanf`)
* Estruturas condicionais (`if` e `else`)
* Estruturas de repetição (`for`)
* Operações matemáticas
* Vetores
* Cálculo de fatorial
* Verificação de números primos
* Sequências numéricas

## 📂 Lista de exercícios resolvidos

1. Exibir a mensagem: **"Olá aluno hoje é sexta-feira"**
2. Exibir mensagem de boas-vindas com o nome do usuário
3. Mostrar o número digitado pelo usuário
4. Mostrar o valor oposto de um valor lógico
5. Realizar a subtração entre dois números
6. Imprimir os 10 primeiros números negativos
7. Verificar se um número é maior que 10
8. Verificar se um número é positivo ou negativo
9. Dividir o maior número pelo menor
10. Calcular o valor de **Y = X + 5**
11. Calcular a média de três números
12. Calcular o valor de **B** na fórmula **A/5 = B/(A+2)**
13. Imprimir o quadrado de 15 números digitados
14. Gerar os 10 primeiros termos de uma sequência
15. Calcular o fatorial de um número
16. Verificar se um número é primo
17. Imprimir todos os números primos até um número informado
18. Imprimir um vetor com valores específicos

## ⚙️ Tecnologias utilizadas

* Linguagem C

## 📖 Aprendizado

A realização desses exercícios permitiu reforçar conceitos fundamentais de lógica de programação e compreender melhor a sintaxe da linguagem C. A atividade também contribuiu para o desenvolvimento do raciocínio lógico na resolução de problemas computacionais.
